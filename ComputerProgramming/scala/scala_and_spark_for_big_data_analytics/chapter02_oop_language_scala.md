#### 스칼라의 케이스 클래스
> 케이스 클래스란  자동으로  생성되는 메서드를 포함한 인스턴스를 생성할 수 있는 클래스다. 또한 케이스 클래스는 자동으로 생성되는 컴패니언 오브젝트도 포함한다.

----
> 케이스 클래스의 기본 구문

    case class <식별자>([var]<식별자>:<타입>[, ...])[extends <식별자>(<입력파라미터>)] [{ 필드와 메소드 }]

----
>케이스 클래스는 

* 패턴 매칭이 가능하다.
* 다음의 메소드가 이미 구현되어 있다.
  * hashCode
  * apply
  * copy
  * equals
  * toString
  * unapply
